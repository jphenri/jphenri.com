## Getting Started

### Prerequisites

- Ruby
- Bundler
- Jekyll

### Installation

1. **Clone the repository**:
    ```bash
    git clone https://github.com/jphenri/website.git
    cd website
    ```

2. **Install the dependencies**:
    ```bash
    bundle install
    ```

3. **Serve the site locally**:
    ```bash
    bundle exec jekyll serve
    ```

4. **Visit the site**:
    Open your web browser and go to `http://localhost:4000`.
